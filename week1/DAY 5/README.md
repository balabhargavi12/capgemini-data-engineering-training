📊 SQL Practice – NULL Handling & REGEX
🚀 Overview

This project focuses on essential data cleaning techniques in SQL, particularly handling missing values and extracting patterns from text using Regular Expressions (REGEX). It reflects real-world scenarios where raw data is often incomplete or unstructured.

📌 Topics Covered
🔹 NULL Handling
Using IS NULL and IS NOT NULL to identify missing values
Applying ISNULL / IFNULL for replacement
Utilizing COALESCE for fallback logic across multiple columns
Using NULLIF to prevent invalid comparisons
Managing NULL values in arithmetic calculations
🔹 REGEX (Regular Expressions)
Pattern matching in text data
Extracting numeric and textual information
Parsing email addresses
Extracting identifiers (e.g., EMP001, ORD-11)
Handling both structured and unstructured text
⚙️ Key Use Cases
Replacing missing values with appropriate defaults
Preventing divide-by-zero errors in calculations
Extracting meaningful insights from messy datasets
Ensuring safe and accurate computations involving NULL values
📈 Outcomes
Enhanced data cleaning and preprocessing skills
Strong understanding of NULL behavior in SQL
Improved ability to write and apply REGEX patterns effectively
🧠 Key Learnings
COALESCE is highly effective for implementing fallback logic
Proper NULL handling is crucial for reliable data analysis
REGEX requires careful design, testing, and validation
✅ Conclusion

This project strengthens foundational SQL skills required for data engineering and analytics, with a focus on handling real-world data challenges such as missing values and unstructured text.
