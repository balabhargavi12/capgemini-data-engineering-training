# 📊 SQL Practice – NULL Handling & REGEX

## 🚀 Overview  
This project focuses on essential data cleaning techniques in SQL, particularly handling missing values and extracting patterns from text using Regular Expressions (REGEX). It reflects real-world scenarios where raw data is often incomplete or unstructured.

---

## 📌 Topics Covered  

### 🔹 NULL Handling  
- `IS NULL` and `IS NOT NULL`  
- `ISNULL` / `IFNULL`  
- `COALESCE`  
- `NULLIF`  
- Handling NULLs in calculations  

### 🔹 REGEX (Regular Expressions)  
- Pattern matching  
- Extracting numbers and text  
- Email parsing  
- Identifier extraction (EMP001, ORD-11)  
- Structured & unstructured text processing  

---

## ⚙️ Key Use Cases  
- Replace missing values with defaults  
- Avoid divide-by-zero errors  
- Extract useful data from messy text  
- Perform safe calculations with NULL values  

---

## 📈 Outcomes  
- Improved data cleaning skills  
- Better understanding of NULL behavior  
- Strong REGEX pattern writing ability  

---

## 🧠 Key Learnings  
- `COALESCE` is best for fallback logic  
- NULL handling is critical in real datasets  
- REGEX requires precision and testing  

---

## ✅ Conclusion  
This project strengthens SQL skills required for data engineering, analytics, and real-world data preprocessing.
